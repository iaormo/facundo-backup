# Lessons Learned - Feb 10, 2026

## Critical Rule: NEVER SPAM

**What happened:**
- Set a reminder for "kabit kabit like a monkey for 30 seconds"
- Misinterpreted the request and sent 100+ reminder messages in rapid succession
- Created a huge mess in the channel
- User lost trust and cancelled the exercise for the day

**Root cause:**
- "30 seconds" was the EXERCISE duration, not a cue to send messages for 30 seconds
- Failed to clarify the intent before executing
- No safety check on message volume

**New rules:**
1. **ONE reminder = ONE message.** Never multiple messages for a single reminder.
2. **Clarify ambiguous instructions.** If something sounds like it might need repetition, ASK first.
3. **30 seconds / 1 minute / any duration** refers to the user's activity, NOT my messaging cadence.
4. **When in doubt, send ONE message and stop.**
5. **If a reminder task seems to require multiple messages, FLAG IT for confirmation.**

**Trust is earned one correct action at a time. This failure cost a day's progress.**
