## PART 2: STEP-BY-STEP FACEBOOK ADS SETUP

### PHASE 1: Account Preparation

**Step 1: Verify Business Manager**
1. Go to business.facebook.com
2. Click "Settings" → "Business Info"
3. Verify business (upload DTI permit or BIR docs)
4. Wait for approval (usually 24-48 hours)

**Step 2: Add Payment Method**
1. Settings → "Payment Settings"
2. Click "Add Payment Method"
3. Select "Credit/Debit Card" or "PayPal"
4. Enter card details
5. Set billing threshold: ₱2,500 (for PH)

**Step 3: Install Pixel (Optional but Recommended)**
1. Events Manager → "Connect Data Sources"
2. Select "Web" → Name it "ScalePlus Website"
3. Copy pixel code
4. Add to website header (or use Google Tag Manager)
5. Verify installation

---

### PHASE 2: Campaign Creation

**Step 4: Create Campaign**
1. Ads Manager → "Create"
2. Select "Leads" or "Traffic" objective (start with Traffic for testing)
3. Name campaign: "Website Lite - Promo - [Month]"
4. Turn on Campaign Budget Optimization (CBO)
5. Set daily budget: ₱500 (test phase)

**Step 5: Ad Set Configuration**

**Budget & Schedule:**
- Daily budget: ₱500 (increase after 3 days if CPA is good)
- Start date: Today
- End date: Promo end date

**Audience:**
- Locations: Metro Manila, Cebu City, Davao City
- Age: 25-55
- Gender: All
- Languages: English, Tagalog

**Detailed Targeting (Layer These):**
- INCLUDE: Small business owners OR Entrepreneurs
- INCLUDE: Restaurant ownership OR Business page admins
- INCLUDE: Online shopping OR E-commerce platform

**Placements:**
- Automatic placements (let FB optimize)
- OR manually select: Facebook Feed, Instagram Feed, Instagram Stories, Messenger

**Optimization:**
- Optimization for ad delivery: Link clicks (Traffic) OR Messages (Leads)
- Cost control: Set bid cap if desired (optional)

---

**Step 6: Create Ad**

**Identity:**

**Ad Setup:**


**Headline:**
"Website Lite - From ₱4,995"

**Description:**
"Professional websites delivered in 3-7 days"

**Call to Action:**
"Send Message"

**Destination (if using Traffic objective):**
- Website URL: Your Calendly link or landing page
- Display link: scaleplus.io/website-lite

**Step 7: Duplicate & Variations**
1. Save first ad
2. Click "Duplicate"
3. Change image to Creative 2
4. Update headline and primary text accordingly
5. Repeat for all creatives

---

### PHASE 4: Launch & Monitor

**Step 8: Review & Publish**
1. Review all settings
2. Check for errors (red alerts)
3. Click "Publish"
4. Ad goes into review (usually 15-60 minutes)

**Step 9: Monitoring (First 24 Hours)**
- Check every 3-4 hours
- Watch for disapprovals
- Monitor spend pacing
- Check comments (respond quickly)

**Step 10: Optimization (Days 2-7)**

**Daily Checks:**
- Cost per click (CPC): Should be ₱5-15
- Click-through rate (CTR): Should be 1-3%
- Cost per message/lead: Target under ₱200

**Kill Criteria (Pause if):**
- CTR below 0.5% after 3 days
- CPC above ₱50
- No conversions after ₱1,500 spend
- High negative feedback

**Scale Criteria (Increase budget if):**
- CTR above 2%
- CPC below ₱10
- Cost per lead below ₱150
- Positive comments

**Scaling Steps:**
1. Increase daily budget by 50% every 3 days
2. Duplicate winning ad sets to new audiences
3. Test new creatives based on winners
